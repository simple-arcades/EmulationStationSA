 INSTALLATION
===============================================================================

  1. Place SAStyle.h into es-core/src/ (new file)
  2. Overwrite all 40 modified files in your source tree
  3. Rebuild via Docker as usual

  NOTE: No CMakeLists.txt changes needed — SAStyle.h is header-only.

===============================================================================
 HOW TO CUSTOMIZE
===============================================================================

Open SAStyle.h. Every visual constant is there with comments:

  FONT
    SA_FONT_PATH              — Custom .ttf file path ("" for built-in)

  COLORS (RRGGBBAA hex)
    SA_TITLE_COLOR            — Menu headings
    SA_TEXT_COLOR              — All menu entry text
    SA_SUBTITLE_COLOR         — Secondary/dim text
    SA_GAMENAME_COLOR         — Game name in gamelist views
    SA_SECTION_HEADER_COLOR   — Folder headers in submenus
    SA_VERSION_COLOR          — Version string in main menu
    SA_SCRAPER_SUBTITLE_COLOR — Scraper progress subtitle
    SA_BUTTON_TEXT_FOCUSED    — Button text when highlighted
    SA_BUTTON_TEXT_UNFOCUSED  — Button text when not highlighted
    SA_INPUT_ICON_COLOR       — Controller config icons
    SA_INPUT_MAPPED_COLOR     — Controller config mapped text
    SA_SELECTOR_COLOR         — Highlighted row bar
    SA_SELECTOR_EDGE_COLOR    — Highlight bar edges
    SA_SEPARATOR_COLOR        — Lines between menu rows
    SA_SLIDER_LINE_COLOR      — Slider track line
    SA_SLIDER_TEXT_COLOR      — Slider value readout
    SA_HELP_ICON_COLOR        — Help bar icons
    SA_HELP_TEXT_COLOR         — Help bar text
    SA_POPUP_TEXT_COLOR        — Toast popup text
    SA_LOADING_BG_COLOR        — Loading bar background
    SA_LOADING_BAR_COLOR       — Loading bar fill
    SA_LOADING_TEXT_COLOR      — Loading screen text

  Change a value, recompile, done.

===============================================================================
 WHAT THIS DOES NOT CONTROL
===============================================================================

  - Menu background texture (:/frame.png NinePatch) — replace the PNG
  - Theme-driven overrides — XML themes can override fonts/colors at runtime
  - SVG icons (on.svg, off.svg, arrow.svg, etc.) — replace the SVG files
  - System carousel colors — theme-driven (0xFFFFFFD8 etc. in SystemView)
